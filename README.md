# Mobile Challenge 20240202

## Intro

App to list English words, based on the Free Dictionary API.

> This is a challenge by [Coodesh](https://coodesh.com/)

#### Technologies (Mobile):

- React Native
- Expo
- UI - Native Base

#### How to start:

- Clone this repo
- Jump to the project folder
- cd Invillia-challenge && yarn install
- Run - yarn ios / yarn android

**Required feature 1** - Attempt the following cases:

- As a user, I should be able to view a list of words with infinite scrolling
- As a user, I should be able to view a word, its meanings, and pronunciation
- As a user, I should be able to save the word as a favorite
- As a user, I should be able to remove the word from favorites
- As a user, I should be able to visit a list of words that I have previously viewed

**Required 2** - Cache the results of requests to speed up the response for repeated searches.

**Required 3** - Follow the wireframe for the data listing page. The position of the items can be changed while maintaining the requested functionalities.

**Extra feature 1** - Implement an audio player using, for example, https://responsivevoice.org/api or native resources;

**Differentiator 2** - Use a Dependency Injection tool.
